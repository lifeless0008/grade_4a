# Grade API Service

RESTful API for Grade Management System built with Node.js, Express, and PostgreSQL.

### Grades
- `GET /api/grades` - Get all grades
- `GET /api/grades/:id` - Get grade by ID
- `POST /api/grades` - Create new grade
- `PUT /api/grades/:id` - Update grade
- `DELETE /api/grades/:id` - Delete grade
- `GET /api/grades/stats/:student_id` - Get student statistics

### Grade Inputs
- `GET /api/grade_inputs` - Get all grade inputs
- `GET /api/grade_inputs/:id` - Get grade input by ID
- `POST /api/grade_inputs` - Create grade input
- `PUT /api/grade_inputs/:id` - Update grade input
- `DELETE /api/grade_inputs/:id` - Delete grade input
